# Network Intrusion Detection System using IBM watsonx.ai AutoAI

## 📌 Project Overview
This project presents an AI-powered **Network Intrusion Detection System (NIDS)** developed using **IBM watsonx.ai Studio** and **AutoAI**. The system analyzes network traffic data and classifies it as either **Normal** or one of several cyberattack categories, enabling early threat detection and improved network security.

## 🎯 Problem Statement
Develop a robust Machine Learning-based Network Intrusion Detection System capable of identifying and classifying different types of cyberattacks such as:
- DoS (Denial of Service)
- Probe
- R2L (Remote-to-Local)
- U2R (User-to-Root)

The solution aims to distinguish malicious traffic from normal network activity and support proactive cybersecurity measures.

## 🚀 Project Workflow
1. Upload network traffic dataset to IBM watsonx.ai.
2. Configure an AutoAI experiment.
3. Automatically preprocess data and engineer features.
4. Train and compare multiple Machine Learning models.
5. Select the best-performing Random Forest model.
6. Predict whether network traffic is normal or malicious.
